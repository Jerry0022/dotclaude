# Build-ID Specification

## What it is

A **human-readable identifier** for the current build session.

- **In a worktree**: `worktree-hash` (e.g., `magical-napier-a3f9b21`) — worktree name as prefix, content hash as suffix.
- **Outside a worktree**: 7-char content hash only (e.g., `a3f9b21`).
- Content hash is always computed over source code + assets. Independent of git commits. Same source = same hash (idempotent, deterministic).

Generated by: `node {PLUGIN_ROOT}/scripts/build-id.js`

> **Path rule**: Always use `{PLUGIN_ROOT}/scripts/build-id.js` — never `~/.claude/scripts/build-id.js`.
> The script lives inside the plugin directory, not in a global scripts folder.
> In bash snippets within project skills, use the `$CLAUDE_PLUGIN_ROOT` env variable.

## What it hashes

**Included**: Source code files, assets (images, fonts, templates)
**Excluded**: Config files, docs, lock files, build artifacts, dist/

## When to generate

| Situation | New build-ID? |
|---|---|
| App started (by user or AI) | Yes — display as `Build a3f9b21 gestartet` |
| Feature is testable (user could try it) | Yes |
| Ship flow (Step 2: Quality Gates) | Yes — after successful build |
| Code doesn't compile | No |
| Typo/comment fix, no functional change | No |
| WIP commit, code half-finished | No |

**Rule of thumb**: Could someone start the app and see the difference? Yes = new build-ID.

## Where it appears

| Location | Shown |
|---|---|
| Completion Card title | `## ✅ magical-napier-a3f9b21 · Summary` (worktree) or `## ✅ a3f9b21 · Summary` (no worktree) |
| Console log on app start | `Build magical-napier-a3f9b21 gestartet` or `Build a3f9b21 gestartet` |
| BUILDLOG.md entry | `## a3f9b21 — YYYY-MM-DD` (always hash-only for log permanence) |
| package.json `buildId` field | `"buildId": "a3f9b21"` (always hash-only for reproducibility) |
| README.md | No (version only) |
| Git tags | No (version only) |

## Build Log (BUILDLOG.md)

Every shipped build gets an entry:

```markdown
## a3f9b21 — YYYY-MM-DD
Version: x.y.z
Branch: feat/42-name
PR: #42
Commit: e8f2a1c
Changes:
- Change 1
- Change 2
```

Newest at top. Created retroactively if missing.
